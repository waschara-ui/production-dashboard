⚙️ ตั้งค่า Google Apps Script

วาง URL ของ Google Apps Script Web App เพื่อให้ CCR ส่งข้อมูลเข้า Google Sheet ได้ตรง

วิธีตั้งค่า (ทำครั้งเดียว):
1. เปิด Google Apps Script → Deploy → New deployment → Web App
2. ตั้ง Execute as: Me · Access: Anyone
3. Copy URL แล้ววางในช่องด้านล่าง
4. กด บันทึก
